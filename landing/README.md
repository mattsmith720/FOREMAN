# Foreman marketing site

Separate Vercel project — YC-clean landing register with Foreman safety-orange accent.

```bash
cd landing
npm install
npm run dev          # http://localhost:3001
npm run capture-assets  # refresh screenshots from prod demo
npm run build
```

Deploy:

```bash
vercel --prod
```

See `docs/landing/OPERATOR_CHECKLIST.md`.
